## Exercícios de Lógica de Programação

Este repositório contém exercícios de lógica de programação referentes a lista 02 de atividades.

Os exercícios abordam o uso de estruturas condicionais, fluxogramas e algoritmos em Portugol.

### Exercícios

- **Sensor de Ré**  
  Algoritmo e fluxograma que analisam a distância de um obstáculo para definir o tipo de alerta sonoro emitido pelo sensor.

- **Filtro de Foto**  
  Algoritmo e fluxograma que simulam a aplicação de um filtro em uma foto e a decisão de postar ou salvar a imagem.

- **Saque em Caixa Eletrônico**  
  Algoritmo e fluxograma que verificam o saldo disponível antes de permitir a realização de um saque.
